RT Gallery enables a "Gallery" post type and "Category" taxonomy.


==Description==

This plugin registers a custom post type for galleries and a taxonomy for categories.

Selected themes by Ruven provide templates to display galleries.
In other themes you most likely have to add them by yourself.


==Changelog==

= v1.0 =
* Initial Release.


==Installation==

This plugin can be installed directly from your site:

1. Log in and navigate to Plugins &rarr; Add New.
2. Type "Ruven Themes: Gallery" into the Search input and click the "Search Plugins" button.
3. Locate the "Ruven Themes: Gallery" in the list of search results and click "Install Now".
4. Click the "Activate Plugin" link at the bottom of the install screen.
5. A "Gallery" menu item will appear in the admin menu.

It can also be installed manually:

1. Download the "Ruven Themes: Gallery" plugin from WordPress.org.
2. Unzip the package and move it into your plugins directory.
3. Log into WordPress and navigate to the "Plugins" screen.
4. Locate "Ruven Themes: Gallery" in the list and click the "Activate" link.
5. A "Gallery" menu item will appear in the admin menu.
